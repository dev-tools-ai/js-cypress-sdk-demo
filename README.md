# Getting started

First install dependencies:

```
npm install
```

Then add you API key to `smartdriver.config.js`.

```js
module.exports = {
  apiKey: "<<YOUR_API_KEY_HERE>>",
};
```

Now you can run cypress using:

```
npm run cypress
```
